Welcome to the documentation for Voice Transcriber – a multifunctional Telegram bot built with aiogram v3, designed to transcribe and manage voice messages with multilingual and admin features.

Voice Transcriber is a multifunctional Telegram bot built with aiogram v3. It transcribes voice messages, supports multilingual interaction, and includes admin tools via a user-friendly interface.

⚙️ Dependencies

Python Packages:

aiogram==3.x
SQLAlchemy
pydantic
asyncio

System Requirements:

  • ffmpeg must be installed and accessible in the system path.

🚀 Installation

  1. Clone the repository:
    git clone https://github.com/your-repo/voice-transcriber-bot.git
cd voice-transcriber-bot
  2. Install dependencies:
    pip install -r requirements.txt
  3. Edit config.py:
    BOT_TOKEN = "your_bot_token"
ADMIN = your_telegram_id
  4. Install ffmpeg:
    • Linux: sudo apt install ffmpeg
    • Windows: Download and add to system PATH
  5. Run the bot:
    python bot.py
